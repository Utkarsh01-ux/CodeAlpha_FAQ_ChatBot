# FAQ ChatBot

## 📌 Project Description
FAQ ChatBot is a Java Swing desktop application developed as part of the CodeAlpha Java Programming Internship. The chatbot answers frequently asked questions using keyword matching and provides a simple graphical user interface for interaction.

## ✨ Features
- Java Swing GUI
- FAQ-based chatbot
- Keyword matching
- Send message functionality
- Clear chat button
- User-friendly interface

## 🛠 Technologies Used
- Java
- Swing
- HashMap
- IntelliJ IDEA

## ▶️ How to Run
1. Clone this repository.
2. Open the project in IntelliJ IDEA.
3. Run `ChatBotGUI.java`.
